About

This website by Home Virt Lab will be used for databases including but not limited to hosting and interacting with them.

Software Used

The software that was used for creating, maintaining, and administering the databases were Python, MySQL Server, MAMP, and PhpMyAdmin. A more detailed explanation is in the paragraph below this.

For the database setup, configuration, and administration, this was all done on a Windows machine (virtual machine and/or host). Links of how to do the installation and configuration will be included on the about page. All of the software was free to use and download from publicly accessible websites.

To register for Amazon Web Services (AWS), search for the free tier of AWS. Sign up where the button is shown and provide your details. Amazon will ask that you provide them with a card as a form of payment in the event that you go over the free tier limit. (Most people don't go over the free tier limit.) Unfortunately, Amazon does not send out overage alerts for billing purposes by default. A person has to set up billing or other alerts on their own. I would recommend setting up billing and other alerts to avoid charges. Charges can be avoided by only using the machine when you need it or shutting down for brief periods of maintenance time.

Database Software Explained

MySQL

MySQL is a database software used by many people and companies particularly with web hosting. To install MySQL, one has to download Python and the community version of MySQL off of the corresponding websites.

MySQL Installation & Configuration

MAMP

MAMP includes MyPHPAdmin and Apache. Apache is the web server software used to run the web interface for MyPHPAdmin. During the initial setup of MAMP, it will ask you whether or not you want to change the ports for Apache and MySQL. In most instances, it is not necessary to change the default ports for Apache and MySQL, but in some instances the ports have to be changed. An issue encountered when using MAMP is that the Apache and MySQL servers have to be restarted a few times for them to work. To access MySQL databases, MyPHPAdmin is started by clicking on the 'open start page' button where you will then find a MyPHPAdmin button on the start page and you will be redirected to MyPHPAdmin once you click on it.
