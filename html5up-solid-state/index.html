MySQL Databases

MySQL is an open-source database software that is offered by Oracle for people to download off of their website and use it for personal use. Most commonly, web hosting companies offer it as a feature to their customers as part of their web hosting plans.

PhpMyAdmin

PhpMyAdmin is used to administer MySQL databases and is the GUI version in comparison to a command line being used. If one wants to create a MySQL database using a GUI, they go to the MySQL database wizard to create a database and user(s) with the appropriate access privileges.

Learn more

MAMP

MAMP like other MySQL connecting software includes Apache web server and PhpMyAdmin to be able to access and administer the MySQL databases from a web console. Fortunately, MAMP is available for both Mac OS X (initially) and more recently Windows. It is one of the easier applications to use for the simplicity of the setup when it prompts if the MySQL and Apache ports want to be changed or kept the same by the person setting it up. To start and/or stop the MySQL and Apache servers, the person click the button that says start/stop servers. If settings need to be changed, there is a settings icon that can be clicked on. To access PhpMyAdmin, click on open start page. Sometimes, the MySQL and Apache web servers need to be restarted a few times to be able to work properly.
